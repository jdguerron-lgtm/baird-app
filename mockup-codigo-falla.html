Flujo de integración

Cómo se integra el sistema de códigos de falla en el flujo de garantías existente

Arquitectura de la solución

1

Parsear Excel → JSON estático

src/lib/constants/codigos-falla.ts

3,358 códigos con: número, descripción, familia, sistema, componente, complejidad

Familia derivada del rango numérico (1000-1999 = REFRIGERACIÓN, 2000-2999 = LAVADO, etc.)

2

Componente CodigoFallaSelector

src/components/ui/CodigoFallaSelector.tsx

Usa Fuse.js para fuzzy search en español. Pre-filtra por tipo_equipo → familia.

Si match < 60% confianza → muestra pregunta de aclaración.

3

Integrar en página de diagnóstico (solo garantía)

src/app/tecnico/[token]/diagnostico/[id]/page.tsx

Se ubica entre el textarea de diagnóstico y la selección de complejidad.

El selector auto-sugiere al escribir (con debounce de 500ms).

4

Guardar en triaje_resultado JSONB

src/app/api/diagnostico/route.ts

Nuevos campos en el JSON:

{
  ...campos_existentes,
  codigo_falla: 1156,
  descripcion_falla: "Compresor no arranca",
  familia_falla: "REFRIGERACIÓN",
  sistema_falla: "ELÉCTRICO",
  componente_falla: "COMPRESOR"
}
5

Mostrar en admin detalle

src/app/admin/solicitudes/[id]/page.tsx

Badge prominente con código, descripción, familia, sistema, componente.

  • • Tabla solicitudes_servicio (no necesita migración)
  • • Flow de servicios particulares (no aplica)
  • • Cálculo de tarifas y bonos
  • • Evidencias y completar servicio

Decisiones clave de diseño

→

JSON estático en vez de tabla Supabase

3,358 registros es un dataset chico. Se carga con el bundle, sin round-trip al servidor. Actualizar = re-parsear el Excel.

→

Pre-filtro por familia (tipo_equipo)

Si la solicitud es de "Nevera", solo muestra códigos de REFRIGERACIÓN primero. Reduce ruido y mejora la precisión del match.

→

Pregunta de aclaración con opciones

Cuando hay 3+ códigos con score similar, el sistema muestra opciones tipo quiz para que el técnico precise la falla.

→

No es campo obligatorio (por ahora)

Para no bloquear al técnico si no encuentra el código. Se puede hacer obligatorio después cuando estén cómodos con el sistema.
